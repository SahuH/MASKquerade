# Spring2022-CS766-project-MASKquerade
Repository for the CS766 project named MASKquerade


## Requirements

* Python 3+
* Conda
* Pillow
  ```shell
  pip install Pillow
  ```
* Tensorflow>=2.0
* pytesseract
  ```shell
  pip install pytesseract
  ```
* Tesseract
  ```shell
  brew install tesseract
  ```
